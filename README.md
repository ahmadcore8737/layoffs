# Global Tech Layoffs Analysis

## Project Overview

This project analyzes more than 4,500 technology layoff events from 2020–2026.

The objective is to identify trends in layoffs across industries,
countries,
company stages,
and funding levels while demonstrating a complete data analysis workflow using Python.

## Key Findings

• Retail and Hardware experienced the largest layoffs.

• The United States accounted for the majority of global layoffs.

• 2023 recorded the highest layoffs.

• Post-IPO companies experienced the largest workforce reductions.

• Only around 10% of companies contributed roughly 80% of layoffs (Pareto Principle).
